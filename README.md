# dockerfile-helm

Docker image that contains git, kubectl, helm and landscaper.

## Building

Build using the following commmand

```bash
docker build .
```

## Versions

Following are the versions bundled with docker image having the tag `1.0`

| Application | Version |
| ----------- | ------- |
| Helm        | 2.7.2   |
| Landscaper  | 1.0.12  |
| Kubectl     | 1.9.1   |
| Kops        | 1.8.0   |
